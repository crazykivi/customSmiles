Формат файла "stamps.ini":

1-я строка: Название пакета
2-я строка: NETSPHSMILES_V1
3-я ......: число групп картинок (НЕ штампов)
4-я ......: <число картинок в первой группе> <название группы>
    ......: ....
    ......: <число картинок в последней группе> <название группы>
   
       ...: TextStamp1 SmileIndex1
       ...: TextStamp2 SmileIndex2 
       ...: ...
       ...: TextStampN SmileIndexN 

GIF файлы  должны иметь имя: smile-xx.gif где xx - номер смайла (SmileIndex)

Максимальное количество групп: 16
Максимальное количество смайлов в группе: 40
Максимальное суммарное количество картинок GIF - 300
Максимальная длина текстового штампа - 12 символов (пробелы использовать нельзя!)
